# Верстка сложного лендинга Go-Surf. Пет-проект.

Имеется: 
          
    слайдеры.        
    svg анимации в момент смены слайдов.
    анимации элементов при скролле страницы.
    
    
Зависимости:

    jQuery
    slick-carousel
    wowjs
    
    
Запуск:

    1. npm i
    2. gulp
